# 🌿 Skincare Routine Calendar

A personal, mobile-friendly skincare routine tracker built as a single HTML file — no app download, no account, no ads. Just your routine, always one tap away.

## About

This calendar was built to support a targeted skincare protocol for **combination skin** focused on clearing **acne scarring and post-inflammatory hyperpigmentation (PIH) on melanated skin**.

It includes a full morning routine and a 7-night evening rotation using clinically-backed active ingredients, organized to avoid unsafe product combinations.

## Features

- 📱 Mobile-optimized with tab navigation (Morning / Evening / Notes)
- 🌙 7-night accordion rotation with expandable step-by-step instructions
- ☀️ Daily morning routine with full product breakdown
- 📋 Notes tab with key reminders, product warnings, and a results timeline
- 💾 Single `.html` file — no dependencies, works offline

## Routine Overview

### ☀️ Every Morning
1. Double Cleanse — Klairs Oil → CeraVe Foaming Cleanser
2. The Ordinary Alpha Arbutin 2% + HA
3. COSRX Snail 96 Mucin Essence
4. Topicals Faded Serum
5. LRP Toleriane Double Repair Matte Moisturizer
6. Black Girl Sunscreen SPF 30

### 🌙 Evening Rotation

| Night | Type | Key Products |
|-------|------|-------------|
| 1, 3, 5 | Tretinoin | Niacinamide, Centella Cream, Tretinoin 0.025%, ANR, Vaseline |
| 2, 4 | Azelaic Acid | Alpha Arbutin, Faded Serum, Azelaic Acid 15%, Effaclar Duo |
| 6 | Exfoliation | AHA 30% + BHA 2%, Centella Cream, Vaseline |
| 7 | Rest & Repair | Double Cleanse, Snail Essence, ANR, Vaseline |

## ⚠️ Important Safety Notes

Never combine on the same night:
- Tretinoin + AHA/BHA
- Tretinoin + Vitamin C
- Tretinoin + Benzoyl Peroxide
- Azelaic Acid + AHA/BHA

## What to Expect

- **Weeks 2–6** — Possible tretinoin purge (temporary breakouts)
- **Months 2–3** — Noticeable improvement in brightness and texture
- **Months 4–6** — Significant PIH clearing with consistent SPF use
